# Amireux Theme

拼拼凑凑弄出了一款Typecho博客主题

目前主题只覆盖了最基本的功能，后期会不断完善。

## 主题预览

[暮城留风](https://www.liaocp.cn/)

## 使用方法

1、下载仓库代码压缩包，解压重命名为：`amireux`

2、将文件夹传至站点 `/usr/themes ` 目录下

3、后台启用主题
